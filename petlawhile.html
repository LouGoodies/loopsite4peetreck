Pętla while w PHP

while (warunek) {
... tutaj znajduje się blok kodu wykonywany w pętli ...
}

Kod zawarty w pętli wykonuje się tak długo, dopóki warunek zwraca wartość true.
Gdy warunek nie jest spełniony przy wejściu w pętlę, kod programu zawarty w pętli nie wykona się ani razu.

Przykład użycia pętli while - po wykonaniu dostaniemy wynik "Lubię pizze" 10 razy

                            <?php
                            $i=0;
                            while ($i<=9) {
                            echo "Lubię pizze";
                            $i++;
                            }
                            ?>
