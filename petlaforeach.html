Pętla foreach w PHP

Foreach jest nieco inną pętlą od pozostałych. Służy do operacji na tablicach. Dokładniej mówiąc, możemy dzięki niej łatwo przeglądać kolejne elementy tablicy.
Podstawowy zapis tej pętli wygląda następująco:

                            <?php
                            foreach ($tablica as $element) {
                            }

Za każdym przejściem tablicy, pobierany jest jej kolejny element i przypisywany do zmiennej $element.
Może to być kolejna tablica, obiekt czy pojedyncza wartość, bez znaczenia. foreach automatycznie kończy się, gdy „przejrzy” całą tablicę.
Jak możemy w praktyce użyć tej pętli? Posłużę się tu przykładem z lekcji wprowadzającej zagadnienie tablicy.
Załóżmy, że mamy w tablicy listę imion uczestników wycieczki i potrzebujemy wyświetlić wszystkie. Zrobimy to tak:

Pierwszy przykład użycia pętli foreach - po wykonaniu dostaniemy wynik "Jan, Piotr, Adrian, Marek, Adam,"

                           <?php
                           $people = ['Jan', 'Piotr', 'Adrian', 'Marek', 'Adam'];
                           foreach ($people as $person) {
                               echo $person . ', ';
                           }

Bez użycia pętli musielibyśmy ręcznie wypisać wyświetlenie wszystkich elementów.
A gdyby zdarzyło się, że tablica nie ma dokładnie określonej liczby elementów, to już w ogóle byłby problem.

Drugi przykład użycia pętli foreach - po wykonaniu dostaniemy wynik "JanPiotrAdrianMarekAdam"

                           <?php
                           $people = ['Jan', 'Piotr', 'Adrian', 'Marek', 'Adam'];
                           echo $people[0];
                           echo $people[1];
                           // ...
                           echo $people[4];

Istnieje także możliwość odczytania indeksu/klucza dla każdego elementu poprzez dodanie =>.

Trzeci przykład użycia pętli foreach - po wykonaniu dostaniemy wynik "Mname: Jan, age: 26, address: Kwiatowa 1, email: jan@example.com,"

                           <?php
                           $person = [
                               'name' => 'Jan',
                               'age' => 26,
                               'address' => 'Kwiatowa 1',
                               'email' => 'jan@example.com'
                           ];
                           foreach ($person as $data => $value) {
                               echo $data . ': ' . $value . ', ';
                           }
