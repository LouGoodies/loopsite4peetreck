Pętla for w PHP

Pętla for to najczęściej wykorzystywana pętla przez programistów, a już na pewno przez autora tego kursu. Ma ona następującą składnię:

for (inicjacja zmiennych; wyrażenie testujące ; zmiana wartości zmiennej) {
... blok kodu wykonywany w pętli ...
}

Pierwszy przykład użycia pętli for - po wykonaniu dostaniemy wynik "Lubię placki" 100 razy

                            <?php
                            for ($x=1; $x<1001; $x++) {
                            echo "Lubię placki<br>";
                            }
                            ?>

Drugi przykład użycia pętli for - po wykonaniu liczba 2 zostanie pomnożona przez siebie i swój wynik 20 razy (2, 4, 8, 16... 1048576)

                            <?php
                            for ($i=1; $i<=20; $i++) {
                            echo 2**$i;
                            echo '<br>';
                            }
                            ?>
